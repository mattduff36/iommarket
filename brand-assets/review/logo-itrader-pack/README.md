# iTrader Logo Review Pack

This folder contains a review-only asset pack generated from `public/images/logo-itrader.png`.

## What was done

- Preserved the original artwork, crop, transparency, and composition.
- Generated deterministic `2x` and `4x` master exports.
- Produced review-ready `PNG` and lossless `WebP` variants sized for the current site usages.
- Added a local inspection gallery at `index.html`.

## Important note

This pack is intentionally conservative. It improves delivery quality without redesigning the logo.

## Start here

1. Open `index.html` to inspect the pack visually.
2. Open `manifest.json` for exact filenames and dimensions.
3. Compare `logo-itrader-original-reference.png` against `logo-itrader-master-2x.png` and `logo-itrader-master-4x.png`.
