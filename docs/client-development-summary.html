Most Recent Development Focus

The latest round of work has been less about adding one single page and more about tightening the whole platform: better account tools, clearer selling routes, stronger dealer tools, better payment handling, automatic listing expiry, and a new internal monitoring area for the admin team.

Accounts

Account area became much more complete

Signed-in users now have a clearer personal area with a dashboard, listing history, saved listings, saved searches, profile controls, security actions, and account deactivation.

Sell Flow

Private and dealer journeys are now clearly split

The site now directs people into the right path depending on whether they are listing as a private seller, subscribing as a dealer, or already have a dealer account.

Pricing

Pricing and sign-up now work together better

The pricing page, sign-up page, and subscription flow now feel connected. A new user can choose a plan first, then create an account with that plan in mind.

Lifecycle

Listing expiry, renewal, sold status, and featured upgrades were strengthened

Live listings are automatically checked for expiry, owners can renew expired listings, live listings can be marked as sold, and eligible listings can be upgraded to Featured.

Admin

Payment operations are more practical

Admins can now search payments more easily, filter them by type, refund listing payments, refund the latest dealer subscription payment, and cancel future dealer billing.

Monitoring

New internal monitoring and alerting tools were added

The admin area now includes a monitoring section that captures site issues from the server side, client side, business processes, and payment webhooks, then lets admins review and triage them.

In plain terms: the recent work has moved the product closer to a launch-ready service. The core marketplace was already there; the newest work makes the journeys clearer, safer, and easier to operate.

How The Site Works

The site supports four main audiences: visitors and buyers, private sellers, dealers, and admins. Each group now has a more complete journey through the platform.

Buyer / Visitor

Browse, compare, save, and contact

  1. Arrive on the homepage and search by make, model, category, or region.
  2. Move into search results, where filters stay in the page URL so searches can be shared or revisited.
  3. Switch between grid and list view, keep scrolling for more results, and optionally include sold listings.
  4. Open a listing to view photos, description, specifications, seller details, and similar listings.
  5. Contact the seller without needing an account, or sign in to save favourites and saved searches.
  6. Report a listing if something looks wrong.
Private Seller

Create a listing in a guided 3-step flow

  1. Sign up or sign in, then choose the private seller route.
  2. Enter the listing details, choose the category and region, and complete any category-specific fields.
  3. Upload between 2 and 20 photos.
  4. Review the preview step and confirm the trust declaration.
  5. If a free launch slot applies, submit straight to moderation. Otherwise continue to Stripe checkout.
  6. After payment, the listing is submitted for admin review before going live.
Dealer

Subscribe, manage stock, and build a public profile

  1. Choose a dealer plan from pricing or the sell flow.
  2. Create a dealer profile if one does not already exist.
  3. Pay for the subscription through Stripe.
  4. Use the dealer dashboard to create listings, track status, view slot usage, and manage the public profile.
  5. Buyers can visit the public dealer page, browse live stock, and read approved reviews.
Admin

Moderate, manage, and monitor the platform

  1. View the dashboard for users, listings, revenue, reports, and recent activity.
  2. Moderate listings and move them through the correct lifecycle.
  3. Review reports, payments, subscriptions, settings, analytics, and monitoring alerts.
  4. Refund eligible payments or cancel dealer subscriptions when needed.
  5. Use the monitoring area to review site issues and operational problems.

Suggested Test Journeys

If the client wants to test the platform like a real user, these are the clearest flows to follow.

Journey 1

Buyer discovery and trust journey

  1. Open the homepage.
  2. Use the main search to browse a category or a make.
  3. Refine the results on /search.
  4. Open a listing and check the image gallery, specs, seller panel, and similar listings.
  5. Try the contact form and the report flow.
  6. If signed in, save the listing as a favourite and save the search.
Journey 2

Private seller listing journey

  1. Create an account from the sign-up page.
  2. Choose the private seller route.
  3. Create a listing with 2 or more photos.
  4. Check the preview step.
  5. Go through the payment or free launch submission step.
  6. Confirm the success screen and then check the listing history in the account area.
Journey 3

Dealer sign-up and stock management journey

  1. Open pricing or dealer subscribe.
  2. Select Starter or Pro.
  3. Create the dealer profile if prompted.
  4. Complete subscription checkout.
  5. Open the dealer dashboard, create a listing, and check slot usage and profile completion.
  6. Visit the public dealer page and confirm the public-facing presentation.
Journey 4

Admin operations journey

  1. Open the admin dashboard.
  2. Review pending listings and moderate one.
  3. Open payments and test searching and filtering.
  4. Review a subscription and confirm that refund/cancel tools are visible.
  5. Open analytics and monitoring to confirm operational visibility is in place.

Comparison Against The PRD

The current product is strongly aligned with the MVP direction in the PRD. The main vehicle marketplace journeys are now clearly visible, and several recent changes were specifically aimed at closing the remaining practical gaps.

Well Aligned

Core MVP items now clearly in place

  • Homepage with search, featured stock, dealer spotlight, and marketplace messaging.
  • Search with shareable filters, grid/list view, advanced filters, and infinite scroll.
  • Listing detail pages with gallery, specs, share actions, reporting, favourites, and contact form.
  • Private seller listing wizard with preview, photo limits, moderation, and payment path.
  • Dealer subscriptions, dealer dashboard, public dealer profile, and listing management.
  • Admin areas for moderation, payments, analytics, settings, and issue handling.
Extended Beyond PRD

Useful additions that improve operations

  • A fuller personal account area for normal users, not just sellers and admins.
  • Dealer reviews on public dealer pages.
  • Automatic expiry sweeps and easier renewal tools.
  • Internal monitoring with issue triage, alerts, and admin review screens.
  • Plan-led sign-up so people can choose the right route from the start.
Still Future / Not The Current Focus

Items not presented as live features today

  • Price-drop alerts and saved filter email alerts.
  • 360 imagery, price history, and recently viewed tools.
  • Map view and finance teaser/integration.
  • Bulk upload for dealers.
  • Mobile apps and broader post-MVP expansion features.
Important note: the PRD mentioned optional Sentry-style error monitoring. The current build has moved further than that by adding a built-in monitoring section inside the admin area, with alerting and issue review tools.

Commercial And Process Notes

Current pricing model

  • Private seller listing: £4.99.
  • Featured upgrade: £10 extra.
  • Dealer Starter: £29.99 per month.
  • Dealer Pro: £49.99 per month.
  • Free launch listings can still be offered where the free launch rules allow it.

Key platform rules

  • Listings do not go live immediately. They move through moderation first.
  • Private seller listings require at least 2 photos and can include up to 20.
  • Expired listings are removed from public visibility and can then be renewed.
  • Payments and subscriptions are confirmed after Stripe sends its final confirmation back to the site.
